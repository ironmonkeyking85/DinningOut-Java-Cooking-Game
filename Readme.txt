#  Dining Out — A Java Cooking Game

Dining Out is a fast-paced cooking and restaurant management game developed in Java. Using JavaFX for the interface, the game combines time-management gameplay with a responsive and easy-to-use design.

---

##  Gameplay Overview

In Dining Out, you step into the role of a chef working in a busy kitchen. The objective is to keep up with customer demand while maintaining accuracy and speed.

As you play, you will:

--Prepare ingredients ahead of incoming orders
--Cook meals correctly within time limits
--Serve customers quickly to maximize tips
--Stay organized as the pace of the game increases

The core challenge revolves around multitasking, quick thinking, and managing limited time effectively.

---

##  Features

--A responsive user interface built with JavaFX, including buttons, timers, and simple animations.
--A cooking system that involves preparing ingredients such as chopping, mixing, and cooking.
--A dynamic order queue that becomes more demanding over time.
--A scoring system based on performance and customer satisfaction..
--Multiple scenes, including a main menu, gameplay screen, and results screen.
--A flexible structure that makes it easy to add new levels, recipes, or gameplay features.

---

##  Tech Stack

Language: Java (JDK 17 or newer recommended)
Framework: JavaFX
Build Tools: Maven or manual configuration using the module path
Assets: PNG images and FXML files for layout design
